<h2 id="experiencia-na-vps">Experiência na VPS (hands-on)</h2>

<ul>
  <li><b>Deploy com Docker em VPS:</b> subi e mantive serviços rodando em containers (aplicação, banco e observabilidade).</li>
  <li><b>Operação e troubleshooting:</b> reinício de VM e validação de subida automática dos serviços com <code>docker ps</code>.</li>
  <li><b>Health checks:</b> validação via <code>curl</code> (app, Prometheus e Grafana), distinguindo “tempo de inicialização” vs falha real.</li>
  <li><b>Observabilidade:</b> Prometheus + Node Exporter + Grafana para monitoramento e dashboards.</li>
  <li><b>Gestão de containers:</b> Portainer para administrar containers, portas e status.</li>
</ul>
